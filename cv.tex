\section*{Research Interests}
\begin{itemize}
	\item \textbf{Proof-Carrying Spec-Driven Development:} Developing a coding agent that simultaneously generates code and formal proofs to enable safe and reliable automated software development.
	\item \textbf{Neuro-Symbolic Agent Safeguards:} Integrating formal methods to ensure the safety and trustworthiness of agentic software.
	\item \textbf{Easy-to-Use Specification Language:} Designing and implementing a novel specification language that combines natural and formal languages, providing an intuitive and accessible interface for both developers and AI.
\end{itemize}
